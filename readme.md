# 介绍

这个项目是 `bkengine` 脚本语言文件的解析器的 `开源实现`
编程语言基于 `python3.8.4`

# bkengine的语法规则

语法规则是bkscr 一共有2种大规则,更多细节请查看 [官网](https://bke.bakery.moe/)

1. 宏 类似kag的面向过程,命令式语言
2. parser系统代码 (类似js或者tjs的面向对象语言)

## 词法分析技术介绍

1. 基于正则表达式 + 手动土法炮制字符串替换
2. 基于逐字符的扫描器

## 开发进度

1. [X] 单行宏的独立解析器(词法分析器)
2. [X] 多行宏的独立解析器(词法分析器)
3. [ ] parser代码的独立解析器(词法分析器)
